# Installation

Step 1. Clone the todolist-react repository from github:

<p> todolist-react: </p>

```
git clone https://github.com/ThaiXuanHieu/todolist-react.git
```

Step 2. install package.
```
npm install
```

Step 3. Start project with command

```
npm start
```
